 Latar Belakang
Di era digital, informasi mengenai seminar, workshop, pelatihan, lomba, hingga acara sosial sering kali tidak sampai ke masyarakat luas secara tepat waktu. Promosi masih terbatas melalui poster, grup WhatsApp, atau media sosial tertentu sehingga tidak semua orang bisa mengakses informasi dengan mudah. Hal ini membuat banyak orang, khususnya anak muda dan komunitas, melewatkan peluang untuk mengembangkan diri dan memperluas jejaring.
Oleh karena itu, diperlukan sebuah aplikasi mobile sederhana namun efektif yang berfokus pada kemudahan mencari dan menemukan event. Dengan antarmuka yang simpel dan fitur yang relevan, aplikasi ini membantu pengguna menjelajahi event di sekitar mereka, menandai event favorit, serta mengunggah informasi event untuk berbagi dengan komunitas lain.
Deskripsi Singkat
GoEvent adalah aplikasi mobile yang dirancang untuk mempermudah pengguna dalam menemukan event yang sesuai dengan minat mereka. Aplikasi ini menghadirkan fitur utama berupa pencarian event berdasarkan kategori dan lokasi, serta halaman Home untuk menampilkan daftar event terbaru. Pengguna juga bisa menyimpan event favorit agar tidak terlewat, dan mengelola informasi diri melalui profil. Selain itu, tersedia fitur Upload Event yang memungkinkan komunitas, kampus, maupun UMKM membagikan kegiatan mereka secara lebih luas.
Keunggulan GoEvent terletak pada kemudahan penggunaan dan penyajian event yang terorganisir, sehingga meskipun fiturnya sederhana, aplikasi ini mampu menjadi solusi praktis bagi siapa saja yang ingin tetap terhubung dengan kegiatan edukatif, sosial, maupun hiburan.
Tujuan Pembuatan Aplikasi
Menyediakan platform sederhana yang memudahkan pencarian event lokal.
Membantu pengguna menemukan event sesuai minat melalui fitur pencarian dan favorit.
Menjadi wadah komunitas, kampus, dan UMKM untuk mempromosikan event mereka secara efektif.
Memberikan pengalaman pengguna yang simpel namun efisien dalam menemukan informasi event.
D. Fitur Utama
Home – Menampilkan daftar event terbaru dan populer.
Upload Event – Memungkinkan komunitas atau individu mengunggah event mereka.
Favorit – Pengguna dapat menyimpan event pilihan untuk diakses kembali.
Profil – Menyediakan halaman profil pengguna untuk mengelola data pribadi dan event yang diunggah

F. Flow Aplikasi
Alur penggunaan aplikasi GoEvent dirancang agar sederhana namun efektif dalam membantu pengguna menemukan dan mengelola informasi event. Berikut penjelasan flow tiap fitur utama:
Halaman Home
Saat aplikasi dibuka, pengguna langsung diarahkan ke halaman Home.
Halaman ini menampilkan daftar event terbaru dan terpopuler.
Pengguna dapat melakukan scroll untuk melihat lebih banyak event.
Jika pengguna memilih salah satu event, sistem akan menampilkan halaman Detail Event yang berisi deskripsi, lokasi, tanggal, serta penyelenggara.


Pencarian Event
Dari halaman Home, pengguna dapat menggunakan fitur pencarian.
Pencarian bisa berdasarkan kategori (misalnya seminar, workshop, hiburan) atau lokasi (misalnya kota/kampus tertentu).
Hasil pencarian langsung ditampilkan dalam bentuk daftar event yang sesuai.


Favorit Event
Pada halaman Detail Event, tersedia tombol “Tambah ke Favorit”.
Jika ditekan, event akan tersimpan di halaman Favorit.
Hal ini memudahkan pengguna untuk menyimpan dan mengakses kembali event yang diminati tanpa harus mencari ulang.


Upload Event
Pengguna yang ingin mempublikasikan kegiatan dapat masuk ke menu Upload Event.
Sistem menampilkan form berisi data seperti judul event, deskripsi, kategori, lokasi, dan tanggal.
Setelah form diisi dan dikirim, event akan tersimpan di database dan otomatis muncul di halaman Home serta dapat ditemukan oleh pengguna lain.


Profil Pengguna
Pada menu Profil, pengguna dapat melihat informasi akun pribadi seperti nama, email, dan foto profil.
Di halaman ini juga ditampilkan daftar event yang pernah diunggah pengguna.
Pengguna dapat melakukan pengaturan dasar pada profil mereka.

G. Rancangan Database
Struktur database yang dibutuhkan untuk menyimpan data event, pengguna, dan favorit. 
Tabel User (id_user, nama, email, password, foto_profil).
Tabel Event (id_event, judul, deskripsi, kategori, lokasi, tanggal, id_user).
Tabel Favorit (id_favorit, id_user, id_event).
